# React Native Projects

This repository contains React Native mobile applications.

## Projects

### ZenLoop - Health & Wellness App

A cross-platform mobile application built with React Native and Expo for tracking exercises and wellness activities.

**Features:**
- User authentication (login/register)
- Exercise browsing with search and filters
- Favourites system with persistence
- Dark mode support
- Redux Toolkit state management
- Responsive design

**Tech Stack:**
- React Native + Expo
- TypeScript
- Redux Toolkit
- React Navigation
- AsyncStorage

See the [ZenLoop README](ZenLoop/README.md) for detailed documentation.
